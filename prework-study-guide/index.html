HTML

  • The head element contains information about the webpage.
  • The head element is not visible to users
  • The body elemts are whats available to the user
  • HTML is essentially the structure of the webpage or the foundation

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three types of CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • Inline CSS styles an element directly in the HTML file
  • We can add a selector such as .card to add properties to specific parts with the appropriate label or class

Git

  • git status: checks what branch we are currently
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout (branch name) switches to that branch
  • git add -A will stage all files in the working branch
  • git commit -m will commit the files with the -m portion adds a message
